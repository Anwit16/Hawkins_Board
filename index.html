How it works
Three steps to your private room.
01
Create a room
Pick a name for your room and set a 4-digit PIN. A unique 6-character code is generated for you instantly.
02
Share the code
Send your partner the shareable link or the 6-character code. They enter the PIN to knock — you let them in.
03
Draw together
Every stroke appears on both screens in real time. Multiple pages, undo, export to PNG — all yours.
Features
Everything a private canvas needs.
Nothing it doesn't.
🔒
PIN-protected rooms
Every room has its own 4-digit PIN. Nobody gets in without it. The admin admits guests one by one.
⚡
Real-time sync
Strokes appear on the other person's screen as you draw them. Powered by Firebase Realtime Database.
📄
Multiple pages
Add as many pages as you need. Rename them, delete them, flip between them — all synced.
↩
Per-user undo
Undo only removes your own last stroke. Your partner's work stays untouched.
💡
Message in lights
Type a word and watch the Christmas lights bar spell it out letter by letter — exactly like Stranger Things.
⚡
Morse code mode
Tap the canvas to send Morse pulses. The other person sees ripple rings expanding from the tap point.
🔓
Admin controls
Lock the canvas to read-only, kick users, clear the entire board — admin has full control.
↓
Export to PNG
Download your canvas as a full-resolution PNG any time. Dark background included.
📱
Works on any device
Fully responsive. Draw with your finger on mobile, or use a mouse on desktop. Same experience everywhere.
